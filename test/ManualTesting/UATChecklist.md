# User Acceptance Testing (UAT) Checklist

This checklist is written from the end-user perspective to ensure the application solves real-world productivity needs.

## Scenarios

### 1. Onboarding & Setup
- [ ] I can easily register an account with my email.
- [ ] The dashboard provides a clear overview of where to start.
- [ ] I can update my name and password securely in settings.

### 2. Daily Task Management
- [ ] I can create tasks for my day quickly.
- [ ] I can break complex tasks into subtasks.
- [ ] I can mark tasks as complete and feel a sense of progression.
- [ ] I can delete tasks I added by mistake.

### 3. Productivity & Focus
- [ ] I can use the Focus timer to work on a task without distractions.
- [ ] The timer is accurate and easy to start/stop.
- [ ] My past focus sessions are recorded so I can review my productivity.

### 4. AI Assistance
- [ ] I can open the Copilot to get help breaking down a large project.
- [ ] The AI responses are helpful and formatted neatly.
- [ ] I can easily convert AI suggestions into real tasks in my list.

### 5. Usability
- [ ] The app looks good and is usable on my mobile phone.
- [ ] The search function (Ctrl+K) helps me find things quickly.
- [ ] The application feels fast and responsive; I am not left waiting for pages to load.
